# Next-canil
🐾 Projeto Pet Adoption - Next.js

Este projeto é uma aplicação web desenvolvida com Next.js, onde é possível visualizar diferentes tipos de animais disponíveis para adoção, como cachorros, gatos e peixes, além de realizar buscas.

### Instalação 
`npm install`

### Para rodar o projeto
git clone https://github.com/seu-usuario/seu-repositorio.git
`npm run start-dev`
